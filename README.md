# Tweeter Project

Tweeter is a simple, single-page Twitter clone.


## Getting Started

1. run `git clone https://github.com/jobinism/jarretCoyleTweeter`
2. run npm install
3. run npm run local

## Dependencies

- Express
- Node 5.10.x or above
- Chance

## DevDependencies
- body-parser
- morgan
- nodemon
